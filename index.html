Pathfinding Visualizer
This is a basic pathfinding algorithm project just to show how the pathfinding algorithms work and the shortest path from two different nodes. Walls can be added to force the algortihm to find other routes toward the end node. After the animation, both start and end nodes can be dragged around to find the shortest path between them instantly, without the animation. The process involves setTimeout() method and recursion to handle the animation properly (Using just a loop wonâ€™t make the animation appear). The project is at this link -> https://gansukii.github.io/Pathfinding-Visualizer/ and the Github repository is at https://github.com/Gansukii/Pathfinding-Visualizer
Sorting Algorithm Visualizer
This project was made to visualize how certain sorting algorithms work. The animations were made using setTimeout and setInterval with recursion. The project is at this link -> https://gansukii.github.io/Sorting-Algo-Visualizer-/
Tentaizu Puzzle Solver
Tentaizu is a game similar to minesweeper. Its goal is to find all of the stars hidden in the board using the given clues. The project uses python to find the solution for any valid 9x9 tentaizu puzzle. The github repository of the project is located at -> https://github.com/Gansukii/Tentaizu-Solver

e-BUS (Electronic Barangay Unified system)
e-BUS is our capstone project where I was the full-stack developer using the MERN stack. e-BUS is a web-based system that caters to multiple barangay systems in one platform. It aims to computerize the way of handling barangay processes as well as accommodate the needs of residents. This web-based system contains a lot of features that will solve numerous existing problems being encountered by most barangays, specifically the prolonged barangay processes. The project is being hosted at -> https://capstone-e-bus.herokuapp.com/
